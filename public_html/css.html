Estilo para HTML con CSS

HTML es utilizado para crear la estructura de una página web.

Para darle formato, hay que usar CSS, hay varias formas de usarlo:

Lo más recomendable es separar el archivo html y los estilos en un archivo css externo.

Inline

Usando el atributo "style" en elementos HTML.

<p style="color:red;">TEXTO<p>

Internal

Usando el elemento "style" dentro del elemento "head".

<head>  <style>  p {color:red;}  </style></head>

External

Enlazando la página web a través del elemento "head" a un archivo css de donde se aplicarán los estilos a nuestro archivo html.

<head><link rel="stylesheet" href="hoja_de_estilos.css"></head>

Atributos

Usar siempre nombres de atributos en minúsculas.

Se puede escribir un atributo sin englobarlos entre comillas, pero es recomendable usarlas siempre.

Para poner comillas dobles ("") dentro de un atributo, se tiene que englobar entre comillas simples(' ') y al revés igual; si queremos mostar comillas simples, se engloba entre comillas dobles.

id

Identifica únicamente a un elemento.

Se usa para referirse a un elemento al cuál le das estilo tanto desde el elemento "head" o desde un archivo externo css.

<p id="parrafoEjemplo1">TEXTO<p>

#parrafoEjemplo1{color:red; }

class

Identifica de forma única a un conjunto de elementos que tengas este atributo.

Se usa para referirse a un conjunto de elementos a los cuales le das estilo tanto desde el elemento "head" o desde un archivo externo css.

<p class="parrafoEjemplo2">TEXTO<p>

<p class="parrafoEjemplo2">OTRO TEXTO<p>

.parrafoEjemplo2{color:red; }

Atributos de style

background-color

Utilizado para darle color de fondo a un elemento.

<p style="background-color:cyan;">TEXTO<p>

Soy un elemento con un fondo de color azul.

font-family

Utilizado para asignar una fuente específica a un texto.

<p style="font-family:verdana;">TEXTO.</p>

Soy un texto con tipo de fuente verdana.

Para tener fuentes de Google Fonts en tu página web debes:

  • Enlazar tu página desde el elemento <head>, con la dirección de la fuente que quieras.

<link href="https://fonts.googleapis.com/css?family="TIPO FUENTE"&display=swap" rel="stylesheet">

  • Ponersela a un elemento con texto.

<p style="font-family: 'TIPO FUENTE', serif;">TEXTO.</p>

Tengo una fuente diferente importada a través de un enlace a google fonts.

font-size

Utilizado para cambiar el tamaño del texto.

<p>style="font-size:200%;">Soy un texto ampliado a un 200%.</p>

Soy un texto ampliado a un 200%.

text-align

Utilizado para alinear texto de un elemento.

<p style="text-align: center;">Soy un texto centrado.</p>

Soy un párrafo centrado

color

Utilizado para darle color al texto de un elemento.

<p style="color: red">Texto rojo<p>

Soy un texto en rojo.

Atributos de color

rgb(a)

Da color a un texto o un fondo con el sistema RGB.

<p style="color: rgb(20, 170, 150);">Texto coloreado con rgb<p>

  • El rango va de 0 a 255 en cada número.

Soy un texto coloreado con rgb.

<p style="color: rgba(20, 170, 150, 1);">Texto coloreado con rgba<p>

  • El cuarto número indica la transparencia del color. El rango va de 0 a 1.

Soy un texto coloreado con rgba.

#

Da color a un texto o un fondo con el sistema HEX.

<p style="color: #36F2C1";>Texto coloreado con #<p>

  • El rango va de 00 a FF en cada par de números.

Soy un texto coloreado por HEX.

hsl(a)

Da color a un texto o un fondo con el sistema HSL.

<p style="color: hsl:(150, 100%, 60%);">Texto coloreado con hsl<p>

  • El rango va de 0 a 360 en el primer número, siendo 0, rojo, 120, verde y 240, azul.
  • De 0% a 100% en el segundo (indica la saturación del color).
  • El tercero va de 0% a 100% (indica el brillo del color).

Soy un texto coloreado por hsl.

<p style="color: hsla(135, 100%, 60%, 0.9);">Texto coloreado con hsla<p>

  • El cuarto va de 0 a 1 (indica la transparencia del color).

Soy un texto coloreado por hsla.
